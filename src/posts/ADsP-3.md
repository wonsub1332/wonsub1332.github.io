### 049 R의 데이터 형 - 벡터(vector)

- 벡터의 생성
    - `1:5`  : 1부터 5까지의 수를 가진 벡터 ⇒ 결과값 : [1, 2, 3, 4, 5] ⭐️
    - `c(1, 3, 4, 6, 9)` : 나열된 겂으로 구성된 벡터 ⇒ [1, 3, 4, 6, 9] ⭐️
    - `seq(1, 5)` : 1부터 5까지 1씩 증가하는 수를 가진 벡터 ⇒ [1, 2, 3, 4, 5]
    - `seq(10, 50, 10)` : 10부터 50까지 10씩 증가하는 수를 가진 벡터 ⇒ [10, 20, 30, 40, 50]
    - `req(3, 5)` : 3을 5개 가진 벡터 ⇒ [3, 3, 3, 3, 3]
    - `req(2:4, each=2)` : ⇒ [2, 2, 3, 3, 4, 4]
    - `req(1:2, 2, 3)` : [1, 2]를 2개 가진 벡터 [1, 2, 1, 2]에 each=3이 적용된 벡터 (세번째 숫자는 each=3 또는 3으로 지정 가능) ⇒ [1, 1, 1, 2, 2, 2, 1, 1, 1, 2, 2, 2]
- 벡터의 연산 `과거 빈출`
    - NA(결측치)는 연산의 대상이 아니다. NA를 포함한 데이터에 대한 평균, 합계 함수 적용 결과는 NA가 된다
    - 데이터의 개수가 서로 다른 두 벡터를 연산할 경우 개수가 작은 것이 반복 사용되며, 경고 메시지와 함께 결과가 표시된다
    - 다양한 기본형을 포함한 벡터를 병합하면 표현범위가 더 넓은 것으로 형식이 변경된다
    - 문자형 > 숫자형 > 논리형 순서로 표현 범위가 넓다
    - 예
        
        <aside>
        💡 (1)
        
        `y ← c(2, 4, NA)`
        
        `y * 3`
        
        결과 : [6, 12, NA]
        
        NA는 연산의 대상이 아니어서 그대로 유지된다
        
        </aside>
        
        <aside>
        💡 (2)
        
        `x ← c(1, 2, 3)`
        
        `y ← c(”apple”, “kiwi”)`
        
        `xy ← c(x, y)`
        
        결과 : [ “1”, ”2”, ”3”, ”apple”, ”kiwi”]
        
        벡터는 한 가지 타입으로만 구성되어야 하므로 더 넓은 표현 범위를 갖는 문자형으로 통일된다
        
        </aside>
        
        <aside>
        💡 (3)
        
        `x ← c(1, 2, NA)`
        
        `mean(x)`
        
        결과 : NA
        
        NA를 포함한 벡터에 대해 mean() 함수의 결과는 NA다
        
        </aside>
        
        <aside>
        💡 (4)
        
        `x ← c(1, 2, 3)`
        
        `y ← c(1, 5)`
        
        `x + y`
        
        결과 : (2, 7, 4)
        
        x와 y 요소의 개수가 다르므로 적은 개수의 요소를 갖는 y가 반복 사용된다
        
        </aside>
        
- 벡터의 인덱싱
    - 인덱싱은 데이터에 부여된 순서 번호인 index를 사용하여 원하는 데이터만 추출하는 작업을 의미한다
    - index는 1부터 시작하여 1씩 증가하는 정수이다
        - v[n] / v[-n] / v[start:end] 와 같이 사용가능
        
        ```r
        v1 <- 3 : 7 #3부터 7까지의 벡터를 생성 [3,4,5,6,7]
        v1[3] # => 5
        v1[-3] # => [3,4,6,7]
        v1[2:3] # => [4,5]
        v1[3:2] # => [5,4]
        v1[c(2,3)] # => [4,5]
        names(v1) <- c('A','B','C','D','E') #요소에 이름 부여
        v1['C'] #=> 5
        ```
        

### 050 R의 데이터 형 - matrix data frame

- matrix
    - 데이터의 형태가 2차원으로 행(row)과 열(column)로 구성된다
    - [수치/문자/논리] 중 하나의 데이터 유형으로만 구성할 수 있다
    - matrix, rbind, cbind 등의 함수를 사용해 생성할 수 있다
    - matrix 생성
        
        ```r
        matrix(data, nrow=1, ncol=1, byrow=FALSE) 
        					#행, 열의 수를 지정하여 데이터를 2차원의 matrix 구조로 만든다
        ```
        
        ```r
        m1 <- matrix(c(1,2,3,4,5,6), ncol=3, byrow=TRUE)
        	# ncol=3 : 열의 수를 3으로 지정한 것이며 데이터가 6개이기 때문에 행의 수는 2가 된다
        	# byrow=TRUE : 1행을 모두 채우고 2행을 채우는 순서로 채우기를 진행하는 것을 의미
        	#      [,1] [,2] [,3]
        	# [1,]   1    2    3 
        	# [2,]   4    5    6
        ```
        
        ```r
        m2 <- matrix(c(1,2,3,4,5,6), nrow=3)
        	# nrow=3 : 행의 수를 3으로 지정한것이며, 데이터가 6이기 떄문에 열의 수는 2
        	# byrow=FALSE : 기본설정으로 별도의 지정이 없는 경우 사용된다 
        	#      [,1] [,2] 
        	# [1,]   1    2  
        	# [2,]   3    4 
        	# [3,]   5    6
        ```
        
    - rbind
        
        ```r
        rbind(vectors or matrices)
         #rbind 함수에 목록으로 제공된 vector나 matrix들이 각각 행방향으로 연결된 행렬이 생성
        ```
        
        ```r
        x <- 1:3           # [1, 2, 3]
        y <- seq(10,30,10) # [10,20,30]
        
        결과1 <- rbind(x,y) # x가 첫번째 행, y가 두번째 행으로 생성된 matrix에 결과1 부여 
        	#      [,1] [,2] [,3]
        	# [1,]   1    2    3 
        	# [2,]  10   20   30
        
        ```
        
    - cbind
        
        ```r
        cbind(vectors or matrices)
         #cbind 함수에 목록으로 제공된 vector나 matrix들이 각각 열방향으로 연결된 행렬이 생성
        ```
        
        ```r
        x <- 1:3           # [1, 2, 3]
        y <- seq(10,30,10) # [10,20,30]
        
        결과2 <- rbind(x,y) # x가 첫번째 열, y가 두번째 열으로 생성된 matrix에 결과2 부여 
        	#      [,1] [,2] 
        	# [1,]   1   10  
        	# [2,]   2   20 
        	# [3,]   3   30
        ```
        
    - matrix 인덱싱
        - matrix의 이름 뒤에 대괄호를 붙인 뒤, 행 번호, 열번호를 (,)로 구분해 지정할 수 있음
        - 행 번호 및 열 번호는 1개의 숫자이거나, 벡터의 형태일 수 있음
        
        ```r
        m <- matrix(c(1,2,3,4,5,6),nrow=2, byrow=TRUE)
        	#      [,1] [,2] [,3]
        	# [1,]   1    2    3 
        	# [2,]   4    5    6
        ```
        
        - `m[1, ]` : 1번행 전체 ⇒ [1, 2, 3]
        - `m[ , 1]` : 1번 열 전체 ⇒ [1, 4]
        - `m[2, 1]` : 2번 행 1열 ⇒ 4
        - `m[1, c(2,3)]` : 1번 행  2,3열 ⇒ [2,3]
- data.frame
    - 엑셀의 워크시트와 같은 구조의 2차원 데이터
    - ‘여러가지 데이터 유형’을 가질 수 있고, 벡터별로 다른 데이터 유형 사용이 가능함
    - 벡터가 모여 데이터 프레임을 구성함
    - data.frame에 있는 하나의 vector인 열을 추출하기 위해서는 다음과 같은 명령어를 사용함
        
        `data.frame이름$열이름`

    - data.frame의 이름을 Student라고 하면 korean 열을 추출하기 위해서는 Student$korean을 사용함
    - 4개의 vector가 1개의 data.frame을 구성하며, name, gender는 문자형, korean, english는 숫자형인 것을 알 수 있음

### 052 그래프 종류 `그래프 해석, 구분 ⭐️`

- 그래프의 결과가 통계학적인 유의미를 갖는 것은 아님!
- 상자 그림 (boxplot) `⭐️⭐️⭐️`
    
    ![image.png](/assets/ADsP/part3/image%206.png)
    
    - Min, Q1, Median, Q3, Max 값과 이상값(Outlier) 확인 가능
    - Min, Max : 일반적인 범위(이상치 범위 안쪽)에서 가장 작은 값과 큰 값 의미
    - 따라서 Min, Max 값까지 연결된 선(=수염)의 길이는 대칭이 아닐 . 수있음
    - Q1, Q3 : 각각 제 1사분위수, 제 3사분위수를 의미하며 25%, 75% 위치
    - IQR(사분위수 범위) = Q3 - Q1
    - 이상값 기준 : 하한보다 작은 값이나 상한보다 큰 값
        
        하한 : Q1 - 1.5 * IQR, 상한 : Q3 + 1.5 * IQR
        
    - 이상값으로 그래프에서 동그라미로 표시됨
    - Median : 중앙값으로 50% 위치
    - 상자그림에 평균, 분산, 데이터의 개수 등의 정보는 없음
- 상자그림(boxplot)의 예 : 사료 첨가물(feed)에 따른 닭의 무게(weight)에 대한 `상자그림 해석`

    - casein을 섭취한 닭의 무게가 다른 사료를 섭취한 닭보다 크다

### 054 결측치와 이상값 처리

1. 결측치 (Missing Value) 처리
    - 단순 대치법(Single Imputation)
        - 완전히 응답한 개체 분석
            - Completes case analysis, 불완전 자료는 모두 무시하는 방법
            - 부분적으로 관측된 자료를 무시하므로 생기는 효율성 상실, 통계적 추론의 타당성 문제 존제
        - 평균대치법
            - 관측 또는 실험을 통해 얻어진 데이터의 평균으로 결측값을 대치하는 방법
            - 비조건부 평균 대치법 : 관측 데이터의 평균값으로 대치하는 방법
            - 조건부 평균 대치법 : 회귀분석을 활용하여 대치하는 방법
        - 단순확률 대치법
            - 평균대치법에서 추정한 표준오차의 과소추정문제를 보완하고자 고안됨
            - 추정량의 표준오차 계산 자체가 어려운 문제가 있음, Hot-deck, nearest neighbor 방법 등이 있음
    - 다중 대치법(Mutiple Imputation)
        - 단순 대치법을 1번이 아닌 m번 수행하여 m개의 가상적 완전 자료를 만듦(m은 여러번을 의미함)
        - 통계적 추론에 사용된 통계량의 효율성 및 일치성 등의 문제를 부분적으로 보완 하였으나, 추정량 표준오차의 과소추정 또는 계산의 난해성 문제 등을 가지고 있음
2. 이상값(Outlier) 처리 `⭐️⭐️`
    - 이상값 개요
        - 일반적인 다른 값과 멀리 떨어져 있는 값(거리, 밀도)
        - 분포를 왜곡할 수 있으나 실제 오류인지 통계적으로 검중할 수 없으므로 제거 여부는 해당 분야의 전문가 또는 실무자와 상의하여 판단하여야 한다
        - 반드시 제거해야 하는 것은 아니며, 분석의 목적이나 종휴에 따라 적절한 판단이 필요
        - 이상값도 분석 대상이 될 수 있어 무조건 삭제는 안된다
        - 이상값을 활용한 시스템에는 ‘부정사용 방지 시스템’ 등이 있음
    - 이상값 판단
        1. ESD(Extreme Studentized Deviation) 방법
            - 정규분포에서 나온 표준 값일 경우, 평균과 표준 편차를 사용하여 이상값을 탐색하기 위한 통계적인 방법
            - 평균으로부터 T 표준편차(Standard Deviation)이상 떨어져 있는 값들을 이상값으로 판단하고 T는 3으로 설정하는 이상값 검색 알고리즘
            - $\mu - 3\sigma < 정상데이터 < \mu + 3\sigma$            #   $\mu$:평균, $\sigma$: 표준편차
        2. 사분위 방법
            - Q1, Q3 및 IQR을 사용해서 이상값을 탐색하는 방법으로 상자그림에서 사용됨
            - $(Q1-1.5 \times IQR) < 정상데이터 <(Q3 +1.5 \times IQR)$
            - $IQR=Q3-Q1$   (IQR : 데이터의 가운데 50%, Interquartile Range)
        3. Summary 함수
            - summary 함수의 결과로 표시되는 평균, 중앙값, 최솟값, 최댓값, Q1, Q3등을 보고 판단하는 방법

### 055 통계 분석 개요

1. 통계 관련 기본 용어
    
    ![image.png](/assets/ADsP/part3/image%209.png)
    
    모집단과 표본
    
    1. 모집단(Population)
        
        잘 정의된 연구 목적과 이와 연계된 명확한 연구 대상 데이터 전체 집합
        
        예) 대통령 후보의 지지율을 구할 때의 유권자 집합을 모집단으로 볼 수 있음
        
    2. 모수(Parameter)
        1. 모집단의 특성을 나타내는 수치
        2. 모집단의 평균($\mu$), 분산($\sigma^2$) 같은 수치들을 모수라고 함
    3. 표본(Sample)
        1. 모집단의 데이터를 전부 조사하기 힘들때 모집단에서 추출(Sampling)한 것
        2. 추출한 표본으로 모집단의 특성을 추론(Inference)함 → 오차발생
            
            예) 각종 여론 조사에 참여한 유권자
            
    4. 통계량(Statistic)
        1. 표본의 특성을 나타내는 수치
        2. 표본의 평균($\overline{x}$), 분산($s^2$)과 같은 수치를 통계량이라고 함
2. 확률적 표본추출법의 종류
    1. 단순 무작위추출 (Simple random sampling)
        1. 모집단의 각 개체가 표본으로 선택될 확률이 동일하게 추출되는 경우
        2. 모집단의 개체 수 N, 표본 수 n 일 때 개별 개체가 선택될 확률은 n/N 임
    2. 계통 추출(Systematic sampling)
        1. 모집단 개체에 1, 2,…, N 이라는 일련 번호를 부여한 후, 첫번째 표본을 임의로 선택하고 일정 간격으로 다음 표본을 선택함
            
            예) 1 ~ 100 번호 부여 후, 10개 선택한다면, [1, 11, 21, 31,…, 91] 선택
            
    3. 층화 추출(Stratified sampling)
        1. 모집단을 서로 겹치지 않게 몇 개의 집단 또는 층(strata)으로 나누고, 각 집단 내에서 원하는 크기의 표본을 단순 무작위추출법으로 추출함
        2. 층 : 성별, 나이대, 지역 등 차이가 존재하는 그룹 → `층간에 차이가 있음`
    4. 군집 추출 (Cluster sampling)
        1. 모집단을 차이가 없는 여러 개의 집단(cluster)으로 나누고, 이들 집단 중 몇 개를 선택한 후, 선택된 집단 내에서 필요한 만큼의 표본을 임의로 선택함 → `그룹간에 차이가 없음`
            
            예) 유치원 어린이에 대해 연구할 때, A유치원, B유치원, …과 같이 여러 유치원 중에서 몇 개를 표본으로 추출함
            
    
    <aside>
    💡
    
    비확률 표본 추출법은 특정 표본이 선정될 확률을 알 수 없어 통계학에서 사용할 수 없음
    
    </aside>
    
3. 표본 추출 관련 오차의 종류 및 특징 `어떤 문제점이 있고 어떻게 해결하는지 암기`
    1. 표본 오차/표본 추출 오차 (Sampling Error)
        1. 모집단을 대표하지 못하는 표본을 추출하여 발생하는 오차
        2. 모집단을 전수 조사하는 것이 아니라 표본을 추출하기 때문에 발생하는 오차
        3. 표본 오차는 표본의 크기가 작아짐, 전수 조사에는 0이 됨
    2. 비표본 추출 오차 (Non-Sampling Error)
        1. 표본 오차를 제외한 조사, 집계, 분석 과정에서 발생할 수 있는 모든 오차
        2. 예) 설문/측정 방식이 잘못됨, 자료기록 및 처리의 오류, 무응답/오응답 오류 등
        3. 비표본 추출 오차는 표본의 크기에 비례하여 커짐, 표본의 크기가 크다고 반드시 좋은 것은 아님
    3. 표본 편의(Sampling Bias)
        1. 표본추출방법에서 기인하는 오차로, 모집단을 대표하지 못하고 일부 특정 그룹 또는 특성을 과도하게 반영하는 경우를 의미
        2. 예) 일반인으로부터 피험자를 모집했을 때, 참여자들은 내용에 관심이 높은 사람일 가능성이 있어 표본이 치우칠 수 있음
        3. 확률화(Randomization)에 의해 최소화하거나 없앨 수 있음

### 056 데이터의 분류

1. 데이터 특성에 따른 분류
    1. 범주형 (categorical):
        1. 고유한 값이나 범주 수가 제한된 데이터(문자, 숫자 등으로 표현)
        2. 순서가 없는 명목형, 순서가 존재하는 서열(순위)형이 있음
    2. 수치형(numeric) : 숫자를 사용하여 표현된 데이터 / 정수로 표현되는 이산형과 실수로 표현되는 연속형이 있음
        1. 이산형 (discrete)
            - 셀 수 있고 특정 값과 값 사이에 다른 값이 존재하지 못하는 정수 데이터
                - 예) 주사위 눈의 수, 자녀 수, 사고 횟수, 제품의 개수 등
            - 이산형이면서 종류가 제한된 경우 범주형과 같은 의미를 갖음
                - 예) 주사위 눈의 수(1,2,3,4,5,6) 성별을 나타내는 숫자(1,2,3,4)등
            - 이산형이면서 종류가 제한되지 않은 경우는 연속형과 같은 의미로 사용
                - 예) 사고횟수, 제품의 개수, 판매량 등
        2. 연속형(continuous)
            - 데이터 값과 값 사이에 무수히 많은 다른 값들이 존재하는 실수 데이터
            - 등간척도, 비율척도를 포함함
                
                예) 키, 몸무게, 기온 등
                
2. 척도의 종류  `특징 &  예시 ⭐️` 
    1. 명목척도 (Nominal scale) → 범주형, 순서 X
        1. 단순히 측정 대상의 특성을 분류하거나 확인하기 위한 목적
        2. 숫자로 바꾸어도 그 값이 크고 작음을 나타내지 않고
            
            예) 성별(F,M) , 혈액형(A, B, O, AB), 출생지, 부서 등
            
    2. 서열(순위) 척도 (Oridinal scale) → 범주형, 순서 O
        1. 대소 또는 높고 낮음 등의 순위만 제공할 뿐 양적인 비교는 할 수 없음
        2. 항목들 간에 서열이나 순위가 존재
            
            예) 학점(A,B,C,D,E), 메달(금,은,동), 만족도 (1,2,3,4,5, Likert 척도) 등
            
    3. 등간 척도(구간 척도)(Interval scale) ⇒ 연속형
        1. 순위를 부여하되 순위 사이의 간격이 동일하여 양적인 비교가 가능함(덧셈, 뺄셈 가능)
        2. 절대 0점이 존재하지 않음, 예) 온도계 수치, 물가지수
    4. 비율척도 (Ratio scale)  ⇒ 연속형
        1. 절대 0점이 존재하여 측정값 사이의 비율 계산이 가능한 척도( 사칙연산 가능 )
            
            예) 몸무게, 나이, 형제의 수, 직장까지 거리
            
    
    <aside>
    💡
    
    절대 0점 : 없음을 의미함(무)
    
    - 온도의 0은 상대 0점으로 없음이 아니라 영상, 영하의 중간 지점을 나타냄
    </aside>
    

### 057 집중화 경향 측정

1. 집중화 경향
    - 집중화 경향(Central Tendency) 측정에 사용되는 값들 `각각의 값들을 구할수 있어야 한다`
        - 평균(Mean) / 산술 평균 :
            - 값 들의 무게 중심이 어디인지를 나타내는 값
            - 대칭(Symmetric)일 때는 중앙값과 같은 위치이지만, 한쪽 꼬리가 긴 경우는 평균의 위치가 꼬리 방향으로 이동함
        - 중앙값(Median) :
            - 자료를 크기 순서대로 배열했을 때, 중앙에 위치하게 되는 값, 이상값에 민감하게 반응하지 않음
        - 최빈값(Mode) :
            - 어떤 값이 가장 많이 관찰되는지 나타낸 값, 주로 범주형 변수에서 사용되는 개념
            - 최빈값은 1개 이상일 수 있음 (없을 수도 있음)
        
        <aside>
        💡
        
        이상값과 평균, 중앙값의 관계
        
        [1,2,3,4,5] ⇒ 평균 3
        
        [1,2,3,4,5] ⇒ 중앙값 = 3, [1,2,3,4,5,6] ⇒ 중앙값 = $(3+4)\div2$  = 3.5
        
        [1,1,1,1,2,2,2,2,3] ⇒ 최빈값 [1,2]
        
        [1,2,3,4,5,6,30] 
        
        - 30은 이상값
        - 중앙값 4, 평균 7.3  → 평균이 이상값인 30에 더 가까움
        - 평균은 중앙값에 비해 이상값이 있는 곡과 가까운 위치에 있게 된다
        </aside>
        
    - 집중화 경향 측정 값의 특징
        
        ![image.png](/assets/ADsP/part3/image%2010.png)
        
        - negative-Skewed : 왼쪽 꼬리가 긴 상태, 오른쪽에 데이터가 치우친 상태
        - Positive-Skewed : 오른쪽 꼬리가 긴 상태, 왼쪽에 데이터가 치우친 상태
        - 이상치 (Outlier) 꼬리가 긴 방향에 있을 수 있음
        
        - Mean = Median = Mode 가 모두 동일한 경우, 대칭(Symmetric) 형태
        - Median > Mean 일 때, 음의 왜도 (negative-Skewed)
        - Median < Mean 일 때, 양의 왜도(Positive-Skewed)
2. 데이터 퍼짐 정도
    - 데이터의 퍼짐 정도 측정 : 데이터 집합이 얼마나 퍼져 있는지를 알아보는데 사용하는 값들  `구할수도 있어야하고 개념도 확실히⭐️`
        - 산포도 (dispersion) :
            - 자료의 변량들이 흩어져 있는 정도를 하나의 수로 나타낸 값
            - 산포도가 크면 변량들이 평균으로부터 멀리 흩어져 있음, 변동성이 커짐
            - 산포도가 작으면 변량들이 평균 주위에 밀집, 변동성이 작아짐
        - 범위 (Range) :
            - 범위 = 최댓값 - 최솟값
                
                예) [1,3,5,7,9] 인 경우 : 범위 = 9-1 = 8
                
        - 사분위수 범위 (Interquartile Range) :
            - IQR = Q3 - Q1
            - 가운데 50% 데이터가 위치하는 범위를 의미, 이상치 검출에 사용되는 수치
        - 편차 (Deviation) :
            - 편차 = 변량 - 평균 (어떤 자료의 변량에서 평균을 뺀 값)
            - 편차의 총합은 항상 0
            - 편차의 절댓값이 클수록 그 변량은 평균에서 멀리 떨어져 있고, 편차의 절댓값이 작을 수록 평균에 가까이 있음
        
        <aside>
        💡
        
        자유도(Degree of Freedom)
        
        <aside>
        💡
        
        $$
        \bar{x}=\frac{1}{n} \sum_{i=1}^nx_i \qquad s^2=\frac{1}{n-1}\sum_{x=1}^n(x_i-\bar{x})^2
        $$
        
        - 표본평균의 자유도 : $n$
        - 표본 분산에서 자유도 : $n-1$
        </aside>
        
        표본의 분산을 구할 때 $n-1$을 사용하면 표본의 분산을 모집단의 분산에 근사하게 불편 추정할 수 있으므로 $n-1$을 자유도라고 합니다
        
        </aside>
        
    1. 평균은 같지만 분산은 다른 두 확률 분포
        
        <aside>
        💡
        
        ![image.png](/assets/ADsP/part3/image%2011.png)
        
        </aside>
        
        1. 평균이 동일한 경우 분산이 크면 값이 더 넓게 분포하고, 분산이 작으면 값이 좁은 범위에 분포한다
        2. 집단 내 분산이 작아질수록 ‘평균의 차이’가 분명해진다
        3. 집단 내 분산이 클수록 ‘집단 간 평균값의 차이’가 무의미해진다
        
    2. 분산, 표준편차
        - 표본 분산($s^2$) Variance :
            - 편차의 제곱의 합을 $n-1$로 나눈것
            - 데이터 집합이 얼마나 퍼져있는지 알아볼 수 있는 수치
            - 평균이 같아도 분산은 다를 수 있음
        - 표본 표준 편파($s$)
            - 자료의 산포도를 나타내는 수치, 분산의 양의 제곱근
            - 평균으로부터 각 데이터의 관찰 값까지의 평균거리

        3. 확률 = (사건이 발생할 경우의 수 ) / (표본공간 경우의 수), 확률 값 범위 : 0 ≤ P(A) ≤ = 1

2. 조건부 확률 (Conditional Probability)
    - $P(A|B)$ : 사건 $B$가 발생했다는 조건 아래서 사건 $A$가 발생할 조건부 확률
    - $P(A|B) = P(A\cap B) / P(B) \quad, 단 \,P(B) > 0$
    - 두 사건 $A,B$ 가 독립사건인 경우 :
        - $P(B|A)=P(B)$
        - $P(A|B)=P(A)$
        - $P(A\cap B)=P(A)\cdot P(B)$
    - 예1) 음주, 비음주에 대한 사고, 무사고에 대한 확률이 다음과 같을 때 P(음주|사고)를 구하면?
        
        
        |  | 사고 | 무사고 |
        | --- | --- | --- |
        | 음주자 | 0.07 | 0.23 |
        | 비음주자 | 0.06 | 0.64 |
        - (음주자의 사고)/(음주자의 사고+비음주자의 사고)
        - $0.07/(0.07+0.06)=0.538$
    - 예2) A사에서 고객들의 햄버거 소스 취향을 조사한 결과 75%는 겨자를 사용하고, 80%는 케찹을 사용하며, 65%는 두가지를 사용한다는 사실을 발견했다. 겨자 사용자가 케찹을 사용할 확률은?
        - 사건A : 고객은 겨자를 사용한다 , 사건B : 고객은 케찹을 사용한다
        - $P(B|A) = P(B\cap A) / P(A) = (둘 다 사용하는 사용자)/(겨자 사용자)= 0.65/0.75=0.867$
3. 사건의 종류 ⭐️⭐️⭐️
    1. 독립사건 `↔ 배반사건과 구분`
        1. A의 발생이 B가 발생할 확률을 바꾸지 않는 사건
        2. 두 사건 A,B가 독립이면
            - $P(B|A)=P(B),\ P(A|B)=P(A),\ P(A\cap B) = P(A)\cdot P(B)$ 성립
            - $P(A\cup B)=P(A)+P(B)-P(A\cap B)=P(A)+P(B)-P(A)\cdot P(B)$
        3. 예) 주사위를 던져서 나오는 눈의 값과 동전을 던져 나오는 앞/뒤가 나오는 사건
    2. 배반사건
        1. 교집합이 공집합인 사건, 한쪽이 일어나면 다른 쪽이 일어나지 않을 때의 두 사건
        2. A와 B가 배반사건이면, $P(A\cap B)=0,\ P(A\cup B)=P(A)+P(B)$
        3. 예) 동전 하나를 던져 앞면 나오는 사건, 뒷면 나오는 사건
    3. 종속사건
        1. 두 사건 A와 B에서 한 사건의 결과가 다른 사건에 영향을 주는 사건
        2. $P(A\cap B) =P(A|B)\cdot P(B)$
        3. 예) 음주와 사고 사건

### 060 이산형 확률분포 `각각 이름, 특징, 기댓값, 분산`

![image.png](/assets/ADsP/part3/image%2015.png)

- 베르누이 분포 (Bernoulli Distribution)
    - 매 시행마다 오직 두 가지의 가능한 결과만 일어난다고 할때, 이러한 시행을 1회 시행하여 일어난 두 가지 결과에 의한  값이 각각 0과 1로 결정되며, 다음을 만족하는 확률 변수 X가 따르는 확률분포
    - $X$가 $1$일 때의 확률을 $p$라고 하고 $X$가 $0$일 때의 확률을 $q$라고 할 때, $p$가 0~1 사이의 값이고 $q$는 $1-p$의 값을 가질 때의 $X$가 따르는 확률분포
    - $P(X=0)=q,\ P(X=1)=p,\ 0\le p \le 1,\ q= 1-p$
    - 모수가 하나이며 서로 반복되는 사건이 일어나는 실험의 반복적 실행을 확률분포로 나타낸 것
    - 기댓값 : $E(x)=p$, 분산 : $V(x)=p(1-p) ,\ x=\{ 0,1\}$
    - 예
        - 동전을 던져서 앞면이 나올 확률 → $p=\frac{1}{2},\ q=\frac{1}{2}$
        - 주사위를 던져서 4의 눈이 나올 확률 → $p= \frac{1}{6},\ q= \frac{5}{6}$
        - 주사위를 던져서 4,5의 눈이 나올 확률 → $p= \frac{1}{3},\ q= \frac{2}{3}$
- 이항분포 (Binomial Distribution) → `베르누이를 여러번 수행`
    - 서로 독립된 베르누이 시행을 $n$회 반복할 때 성공한 횟수를 $x$라고 했을 때,성공한 $x$의 확률분포를 말한다
    - 확률변수 $K$가 $n,p$ 두개의 모수를 가지며, $X$~ $B(n,p)$로 표기
        
        $$
        P(k=x)=f(x;\ n,p)=\binom{n}{x}p^x(1-p)^{n-x}\\
        x:\text{성공횟수,}\ n: \text{반복 수행 횟수,}\ p: \text{성공의 확률}
        $$
        
    - $n=1$ 일 때의 이항분포는 베르누이분포이다
    - 이항분포의 기댓값과 분산
        - 이항분포의 기댓값 : $E(x)=np$
        - 이항분포의 분산 : $V(x)=np(1-p)$
- 기하분포 (Geometric Distribution)
    - 베르누이 시행에서 처음 성공까지 시도한 횟수 $x$를 확률변수로 가지는 분포, 지지집합$(x)=\{1,2,3,…\}$
    - 성공확률 $p$인 베르누이 시행에 대해, $x$번 시행 후 첫번째 성공을 얻을 확률, $X \sim G(p)$로 표기
    - $P(X=x)=(1-p)^{x-1}p(x=1,2,3,...)$
    - 실패 횟수에 대해서는 p$P(Y=x)=(1-p)^xp(x=0,1,2,...)$
        - 예) 실패 3회, 성공을 1회 했다면
         $P(X=4)=(1-p)*(1-p)*(1-p)*p=(1-p)^3*p$
    - 기댓값 : $E(x)=\frac{1}{p}$ 분산 : $V(x)=\frac{(1-p)}{p^2}$
    - 기하분포의 예
        - A 야구선수의 홈런 칠 확률이 5%일 때, 이 선수가 x번째 타석에서 홈런 칠 확률 분포
        
        ![image.png](/assets/ADsP/part3/image%2016.png)
        
        $P(X=5)=(1-p)^4*p =(0.95)^4*0.05=0.040725$
        
- 포아송분포 (Poisson Distribution)
    - 단위 시간이나 단위 공간에서 어떤 사건이 몇 번 발생한 것인지를 표현하는 분포
    - 특정 기간 동안 사건(events) 발생의 확률을 구할 때 사용되며, $X \sim \text{Pois}(np)$ 와 같이 표기함
    - $\lambda$ : 정해진 시간 안에 어떤 사건이 일어날 횟수에 대한 기댓값, $P(X=x)=\frac{e^{-\lambda}\lambda^x}{x!}$
    - 확률은 $x= \lambda$ 에서 최대이며, $x$가 커질수록 $0$에 접근함
    - 기댓값 $E(x)=\lambda$, 분산 $V(x)=\lambda$
    - 포아송분포의 예
        
        
        - 어느 AS센터에서 1시간당 평균 120건의 전화가 온다. 이때 1분 동안 걸려오는 전화 요청이 4건 이하일 확률은?
        - 어느 가게에 1시간당 평균 8명의 손님이 온다. 이때 1시간 동안 손님이 10명 올 확률은?
        
        ![image.png](/assets/ADsP/part3/image%2017.png)
        
- 이산균등 분포 (Discrete Uniform Distribution)
    - 정의된 모든 곳에서 확률분포함수 값이 일정한 분포
    - 확률변수가 $n$개의 값을 가질 수 있다면, $k_i$ 일 확률이 $\frac{1}{n}$임 (예:주사위 던지기)
        - 주사위 던지기의 경우 $n=6$이며, 확률은 $\frac{1}{6}$로 동일함

### 062 연속형 확률부

- t - 분포
    - 정규분포는 표본의 수가 적으면 신뢰도가 낮아짐(n이 30개 미만인 경우)
    - 표본이 적은 경우 예측 범위가 넓은 t-분포를 사용함
    - t-분포는 표본의 개수에 따라 그래프의 모양이 변함
        - 표번의 개수가 많아질수록 정규분포와 비슷하며, 적을 수록 퍼짐
        - 표본의 개수가 적을수록 신뢰도가 낮아지기 때문에 예측범위를 넓히기 위해 옆으로 퍼지게 됨
    - 그래프의 x축 좌표를 t값이라 부르며, t분포표를 사용해 구하고 검정에 사용함
    
    ![image.png](/assets/ADsP/part3/image%2024.png)
    
- 카이제곱 분포 ($\chi^2)$
    - 분산의 특징을 확률분포로 만든 것으로, 카이($\chi$)는 평균 0 ,분산 1인 표준 정규분포를 의미함
    - 0 이상의 값만 가질 수 있으며, 오른쪽 꼬리가 긴 비대칭 모양
    - 표본의 수가 많아지면 옆으로 넓적한 정규분포 형태가 됨
- F-분포
    - 카이제곱분포와 같이 분산을 다룰 때 사용하는 분포
    - 카이제곱과 비슷하게 비대칭 모양이며, 양수만 존재, 표본의 수가 많아지면 1을 중심으로 정규분포 모양이됨
    - 카이제곱분포는 한 집단의 분산, F 분포는 두집단의 분산이 크기가 서로 같은지 또는 다른지 비교하는데 사용함
    - 두 집단의 분산을 나우었을 때 1이면 두 집단의 크기가 같음으로 판단
    - F 검정과 분산분석(ANOVA)에서 사용됨

### 063 통계적 추론의 분류 → `‘모집단‘에 대한 추론을 위한 것`

1. 추론 목적에 따른 통계적 추론의 분류
    1. 추정 : Estimation, 통계량을 사용하여 모집단의 모수를 구체적으로 추측하는 과정
        1. 점추정 : Point estimation , 하나의 값으로 모수의 값이 얼마인지 추측함
        2. 구간 추정 : Interval estimation, 모수를 포함할 것으로 기대되는 구간을 확률적으로 구함
    2. 가설검정 : Testing hypotheses, 모수에 대한 가설을 세우고 그 가설의 옳고 그름을 확률적으로 판정하는 방법론
2. 모집단에 대한 특정 분포 가정 여부에 따른 통계적 추론의 분류
    1. 모수적 추론 : Parametric Inference, 모집단에 특정 분포를 가정하고 모수에 대해 추론한
    2. 비모수적 추론 : Non-parametric Inference, 모집단에 대해 특정 분포 가정을 하지 않음 
3. 모수 처리 방식에 따른 통계적 추론의 분류
    1. 빈도주의(Frequentist)
        1. 확률을 사건의 빈도로 보고, 특정 사건이 얼마나 반복되어 발생하는가를 관찰하고 가설을 세우고 모델을 만들어 검증하는 추론 방법
        2. 모수를 어떤 고정된 상수로 생각함
    2. 베이지안(Bayesian)
        1. 확률을 사건 발생에 대한 믿음 또는 척도로 바라보는 관점으로 고정된 데이터의 관점에서 모수에 대한 신념의 변화를 분석함
        2. 수학적 배경이 까다롭고, 계산량이 많아 구현의 어려움이 있어 예전에는 통계학자들로부터 환영 받지 못했음
            
            → 컴퓨터 발달로 사용이 용이해짐
            
        3. 일어나지 않은 일에 대한 확률의 불확실성의 개념과 사건과 관련 있는 여러 확률을 이용해 새롭게 일어날 사건을 추정하는 방법으로 머신러닝에서 많이 활용됨
        4. 머신러닝 활용 예) 나이브 베이지안 분류(naive bayes Classification)

### 064 추정량(Estimator), 추정값(Estimate)

1. 추정량과 추정값
    
    추정(Estimation) : 표본의 통계량(평균,분산,표준편차)를 가지고 모집단의 모수를 추측하여 결정하는 과정
    
    - 추정량 :
        - 모수를 추정하기 위한 관찰 가능한 표본의 식 또는 표본의 함수
        - $Y=f(x)$에서 $f(x)$에 해당하는 것
    - 추정값 :
        - 표본의 식 또는 함수에 실제 관찰자를 대입하여 계산한 값
        - $Y=f(x)$에서 $Y$에 해당하는 것
2. 좋은 추정량 판단 기준 : 일치성, 비편향성(=불편성), 효율성 `암기⭐️` 
    1. 일치성(consistency) :
        1. 표본의 크기가 커짐에 따라 표본오차가 작아져야 한다
    2. 비편향성,불편성(unbiasedness) :
        1. 추정량의 기댓값이 모수의 값과 같아야 한다(편향=0)
        2. 편향(bias)=추정량의 기댓값-실제값(=모수의값)
    3. 효율성(efficiency) :
        1. 추정량의 분산은 최대한 작아야 한다 (최소분산추정량)
        2. MSE(Mean Squared Error)가 작아야 한다

### 065 통계적 추론 - 추정(Estimation)

1. 점추정 (Point Estimation)
    1. 개요
        1. 통계량 하나를 구하고 그것을 가지고 모수를 추정하는 방법
        2. ‘모수가 특정한 값일 것’이라고 추정하는 것
        3. 예) A과목 전체 수강 학생 중 50명을 뽑아 조사한 결과 기말 점수가 80점 이었다면, 50명 뿐 아니라 나머지 A과목을 수강한 학생들의 점수도 80점 정도로 추정함
    2. 점추정량 구하는 법 `종류 및 간략한 정의`
        1. 적률법, 최대가능도추정법(최대우도법), 최소제곱법이 있음
        2. 적률법 - 표본의 기댓값을 통해 모수를 추정하는 방법
        3. 최대가능도추정법(최대우도법) - 함수를 미분해서 기울기가 0인 위치에 존재하는 MLE(Maximum Likelihood estimator)를 찾는 방법
            
            ![image.png](/assets/ADsP/part3/image%2027.png)
            
        4. 최소제곱법 - 함수값과 측정값의 차이인 오차를 제곱한 합이 최소가 되는 함수를 구하는 방법
            
            ![image.png](/assets/ADsP/part3/image%2028.png)
            
2. 구간추정 (Interval Estimation)
    1. 점추정의 정확성을 보완하는 방법으로 통계량을 제시하는 것은 같지만 신뢰구간을 만들어서 추정함
    2. 신뢰구간(Confidence Interval) : `모수`가 포함되리라고 기대되는 범위
3. 신뢰수준 (Confidence Level)
    1. 모수값이 정해져 있을 때 다수 신뢰구간 중 `모수`값을 포함하는 신뢰구간이 존재할 확률
    2. 신뢰수준 95% 의미 : n번 반복 추출하여 산정하는 신뢰구간들 중에서 평균적으로 95%는 `모수`값을 포함하고 있을 것이라는 의미

4. 신뢰구간
    1. 표준오차(Standard Error)
        
        $$
        \text{표본오차}=\frac{\text{원시데이터의 표준편차}}{\sqrt{\text{평균값 계산에 사용한 데이터 수}}}= \frac{\sigma}{\sqrt{n}} \approx \frac{s}{\sqrt{n}}\\~\\ 
        \bar{x_1}\ \bar{x_2}\ \bar{x_3}\ \text{모집단}(\mu)
        $$
        
        - 표본평균이 모집단의 평균값과 얼마나 차이가 있는지 나타냄
        - 모집단에서 표본(Sample)을 무한 번 뽑아서 각 샘플에서 평균을 구했을 때, 그 평균들의 표준편차를 표준오차라고 함
        - 표준오차는 표본의 크기 $n$이 클수록 작은 값을 갖고, 모평균에 대해 추론할 때 표본평균의 표준오차를 사용함
    2. 표본 오차 (Sampling Error)
        
        $$
        \text{표본오차}=\text{오차한계}=\text{임계값} * \frac{\sigma}{\sqrt{n}}(=표본오차)
        $$
        
        $$
        표준정규분포(Z-분포)에서\ 표본오차=z*\frac{\sigma}{\sqrt{n}}\approx z*\frac{s}{\sqrt{n}}
        $$
        
        - 임계값(Critical Value) : 표준정규분포에서는 $z$값, $t$분포에서 $t$값, 카이제곱분포에서는 $\chi^2$값으로 검정통계량이 임계값보다 더 큰 값을 갖는 경우 귀무가설을 기각하게 됨(가설검정에서 다루는 내용)
        - 표본을 샘플링 할 때, 모집단을 대표할 수 있는 전형적인 구성 요소를 선택하지 못함으로써 발생하는 오차
        - 표본오차는 표본의 크기를 증가시키고, 표본 선택 방법을 엄격히 하여 줄일 수 있음
        - 추정을 할 때 모평균 추정구간의 중심으로부터 최대한 허용할 최대허용오차
        - 추정 문제에서 표본오차를 구하라는 것은 ‘오차한계‘를 구하라는 것과 같음
    3. 신뢰구간의 길이 ⭐️⭐️
        1. 신뢰구간 : 모수가 포함되리라고 기대되는 범위
        2. 99% 신뢰수준에 대한 신뢰구간이 95% 신뢰수준에 대한 신뢰구간보다 길다
        3. 표본의 크기가 커지면 신뢰구간의 길이는 줄어든다ㅇ

            l=2*1.96\frac{\sigma}{\sqrt{n}}\\

            $$

            l=2*2.58\frac{\sigma}{\sqrt{n}}
            $$

### 066 통계적 추론 - 가설검정 `절차 & 용어`

1. 가설검정 (Statistical Hypothesis Testing)
    
    모집단에 대해 가설 설정 후, 표본 관찰을 통해 그 가설의 채택 여부를 결정하는 통계적 추론 방법
    
    - 가설검정의 절차
        
        ![image.png](/assets/ADsP/part3/image%2029.png)
        
        - 검정통계량 > 임계값 → 귀무가설 기각
    - 귀무가설(Null Hypothesis)과 대립 가설(Anti Hypothesis)
        - 귀무가설
            - 가설검정의 대상이 되는 가설
            - 연구자가 부정하고자 하는 가설
            - 알고 있는 것과 같음, 변화, 영향력, 연관성, 효과 없음에 대한 가설
                
                예) 과자의 평균 중량은 80g이다
                
        - 대립가설
            - 연구자가 연구를 통해 입증/증명되기를 기대하는 예상이나 주장
            - 귀무가설이 기각되면 채택되는 가설
            - 알고 있는 것과 다름, 변화, 영향력, 연관성, 효과 있음에 대한 가설
                
                예) 과자의 평균 중량은 80g이 아니다
                
    - 가설검정의 종류 : 귀무가설을 기각하는 영역의 위치에 따라 세가지로 나눔
        - 양측 검정 : 귀무가설을 기각하는 영역이 양쪽에 있는 검정
        - 단측 검정 :
            - 좌측 검정 : 귀무가설을 기각하는 영역이 왼쪽에 있는 검정
            - 우측 검정 : 귀무가설을 기각하는 영역이 오른쪽에 있는 검정
        
        ![image.png](/assets/ADsP/part3/image%2030.png)
        
        확률  분포 그래프
        
        - $X$축 : 검정통계량($t$, $Z$, $F$, $\chi^2$ 통계량), $Y$축 : 확률
        - 분포에 따라 모양이 다름
        - 검정 통계량을 사용하여 p-value(제1종 오류 발생 확률)를 구할 수 있음
2. 제1종 오류, 제 2종 오류 ⭐️⭐️
    1. 제 1종 오류($\alpha$ error)
        1. 귀무가설이 참인데 기각되어 발생하는 오류
        2. $\alpha=P$ (제1종 과오를 범함) $= P(H_0\ 기각 | H_0\ 참)$
    2. 제 2종 오류($\beta$ error)
        1. 귀무가설이 거짓인데 채택하는 오류
        2. $\beta=p$(제2종 과오를 범함) $= P(H_0\ 채택 | H_0\ 거짓)$
    
    | 판단 \ 진실 | $H_0$ 참 | $H_0$ 거짓 |
    | --- | --- | --- |
    | $H_0$ 채택 | 신뢰수준(1-$\alpha$)
    제 1종 오류를 범하지 않을 확률 | 제 2종 오류 ($\beta$) |
    | $H_0$ 기각 | 제 1종 오류($\alpha)$ | 검정력 (1-$\beta$)
    제 2종 오류를 범하지 않을 확률 |

    - 두 가지 오류가 모두 작을 수록 바람직함
    - $\alpha$와 $\beta$, ($1-\alpha$), ($1-\beta$) 는 각각 Trade off 관계
    - 표본의 크기를 증가 시키면 표본분산이 작아지게 되어 \alpha,\beta 발생 확률이 작아지지만 조사비용 증가
    - 제 1종 오류를 범할 확률의 최대 허용치를 특정값(유의수준)으로 지정해 놓고 제 2종 오류의 확률을 가장 작게 해주는 검정 방법을 사용함
    </aside>
    
3. 신뢰수준, 기각역, 유의수준
    1. 신뢰수준 ($1-\alpha$)
        1. 제 1종 오류를 범하지 않을 확률, 연구활동(99%), 일반적(95%), 단순설문조사(90%) 사용
    2. 기각역(Critical Riegion)
        1. 귀무가설을 기각하고 대립(=연구) 가설을 채택하게 되는 영역
        2. 귀무가설이 옳다는 전제하에 구한 검정통계량의 분포에서 확률이 유의수준($\alpha$)인 부분
    3. 유의수준($\alpha$)
        1. 귀무가설이 참인데도 기각시키는 확률(제 1종 오류 발생 확률)의 최대 허용 한계
        2. 유의수준 0.05(5%) : 100번 실험에서 제 1종 오류를 범하는 최대 허용 한계가 5번
        3. $유의수준 + 신뢰수준 = 1$ , $유의수준 = 1 - 신뢰수준$, $유의수준 = \sum기각역$
    
    ![image.png](/assets/ADsP/part3/image%2032.png)
    

1. 검정통계량과 유의확률(p-value)
    1. 검정통계량
        1. 표본 데이터를 사용해 계산되어 가설검정에 사용되는 랜덤변수로 검정통계량을 사용해 귀무가설을 기각할것인지 판별할 수 있음
        2. 귀무가설의 기각 여부에 대해 모수로부터 검정통계량이 얼마나 떨어져 있는지에 따라 판단되며, 많이 떨어져 있을수록 귀무가설을 기각하게 됨 → (검정통계량 > 임계값) 일 때 귀무가설 기각
    2. 유의확률 ⭐️
        1. Probability Value, $0≤ \text{p-value} ≤ 1$
        2. 자유도를 고려한 검정통계량에 관한 확률
        3. 제1종 오류를 범할 확률
        4. 귀무가설의 신뢰구간을 벗어나는 확률(=극단적인 표본값이 나올 확률), 판정이 잘못되었을 확률
        5. 귀무가설을 지지하는 정도로 p-value가 작을 수록 그 지지도가 약하다는 것을 의미함
        6. p-value < $\alpha$(유의수준) 일때, 귀무가설을 기각, 대립가설을 채택함
        7. p-value가 0.05(5%) : 귀무가설을 기각했을 때 기각 결정이 잘못될 확률이 5%라는 것을 의미

### 067 모수적 추론과 비모수적 추론

1. 모수적 추론과 비모수적 추론의 이해
    1. 모수적 추론
        1. 모집단에 특정 분포를 가정하고 분포의 특성을 결정하는 모수에 대해 추론하는 방법
        2. 모수로는 평균, 분산 등을 사용
        3. 자료가 정규분포이면서 등간척도, 비율척도 인 경우 사용
        4. 자료가 등간척도, 비율척도이더라도 정규분포가 아니면 비모수적 추론을 사용함
            
            예) 온도, 물가지수, 몸무게, 자녀수 등의 평균, 분산 등을 추론함
            
    2. 비모수적 추론
        1. 모집단에 대해 특정 분포 가정을 하지 않음(정규분포가 아닌 경우도 사용 가능)
        2. 모수 자체보다 분포 형태에 관한 검정을 실시함
        3. 분포 형태 검정을 위해 중앙값, 순위 등을 사용함
        4. 정규분포가 아닌 경우 또는 표본 수가 적을 때 사용함
        5. 자료가 명목척도, 서열척도일 때 사용함
            
            예) 성별, 혈액형, 만족도, 메달 등의 분포형태 또는 관계를 추론함
            
2. 모수적 추론과 비모수적 추론의 선택
    
    ![image.png](/assets/ADsP/part3/image%2034.png)
    
    n: 표본의 크기
    
    n > 30 :
    
    정규성 검정 없이 모수적 추론을 할 수 있음
    
    10 ≤ n ≤ 30 :
    
    정규성 검정을 한 뒤, 정규성을 갖는 모수적 추론을 할 수 있고, 정규성을 갖지 ㅇ날으면 비모수적 추론 또는 정규성 검정 없이 비모수적 추론이 가능함
    
    n < 10 :
    
    비모수적 추론을 사용함
    

### 068 모수적 추론

1. 모수적 추론
    1. 모수적 추론
        1. 검정하고자 하는 모집단의 분포애 대해 가정을 하고, 그 가정하에서 검정통계량과 검정통계량의 분포를 유도해 검정을 실시함
        2. 가정된 분포의 모수에 대해 가설 설정
        3. 관측된 자료를 이용해 구한 표본 평균, 표본 분산 등을 이용해 검정 실시
    2. 모수적 통계의 전제조건
        1. 표본의 모집단이 정규분포를 이루어야 함
        2. 여러 개의 집단간 차이를 검정하는 경우 집단들이 분산이 같아야 함
        3. 변인(=변수)은 등간척도나 비율척도로 측정되어야 함
        4. 전제 조건을 만족하지 않으면 비모수적 추론을 사용해야 함
    3. 모수적 추론 방법
        1. t-검정(One Sample T test, Paired T test, Two Sample T test), ANOVA 검정, z-검정, F-검정, 카이제곱검정(→ 분산 관련/ 비모수적 추론의 카이제곱은 → 적합성, 동질성 )  등
    4. 모수적 추론 방법의 사용 예시
        1. 한 개의 모 집단의 분산 추론에 사용 : 카이제곱검정 
        2. 두 개의 모 집단의 분산으 비교하기 위한 추론에 사용 : F-검정
        3. 모집단의 평균에 대한 추론에 사용 : Z-검정 , t-검정
2. t-검정 `종류 & 검정 결과 분석` 
    - 한 집단의 평균값이 올바른지, 두 집단의 평균 차이가 있는지를 검증하는 방법으로 t값을 사용함
    - t 값이 커질수록 p-value는 작아지며, 집단간 유의한 차이를 보일 가능성이 높아짐
    - 종류
        - One Sample t-test
            - 단일 표본의 평균 검정을 위한 방법
                
                예) S사 USB의 평균 수명은 20000시간이다
                
        - Paired t-test 대응표본 t-검정
            - 동일 개체에 어떤 처리를 하기 전, 후의 자료를 얻을 때 차이 값에 대한 평균 검정을 위한 방법
                
                예) 매일 1시간 한달 걸으면 2kg이 빠진다 (걷기 수행 전/수행 후)
                
            - 가능한 동일한 특성을 갖는 두 개체에 서로 다른 처리를 하여 그 처리의 효과를 비교하는 방법(matching)
                
                예) X 질병 환자들을 두 집단으로 나누어 A,B 약을 투약해 약의 효과 비교
                
        - Two sample t-test 독립표본 t-검정
            - 서로 다른 두 그룹의 평균을 비교하여 두 표본의 차이가 있는지 검정하는 방법
                
                예) 두 집단의 평균 차이 값이 0이다, 2학년과 3학년의 결석률은 같다
                
    1. ANOVA 검정 (평균에 대한 차이 검정)
        - K개 집단의 평균 차이가 있는지 분산을 사용하여 검증하는 방법
        - “집단 간 분산” / ”집단 내 분산” 기반의 F 분포를 이용해 가설을 검증하며 정규성, 등분산성, 독립성을 가정함
            
            ![image.png](/assets/ADsP/part3/image%2035.png)
            
        - 독립변수 - 범주형, 종속변수 - 연속형
        - 귀무가설 기각 시 구체적인 차이를 파악하기 위해, 사후 검정(Post Hoc test)이 필요함
        - 사후검정 종류
            - 등분산/동일 크기 표본 - Tukey 검정
            - 등분산/다른 크기 표본 - Scheff 검정, Fisher’s LSD 검정 등이 있음
        - 일원분산 분석 (One-way ANOVA)
            - 범주형 독립변수가 한개인 경우 사용
                
                예) 계절별 아이스크림 판매량 평균이 동일하다
                
        - 이원분산 분석(Two-Way ANOVA)
            - 독립변수가 두개 (K-way ANOVA, 범주형 변수가 K개인 경우)
            - 두 변수의 상호작용효과와 각 변수의 주효과를 분석함
            - 주효과 : 각 변수로 일원분산분석한 결과
                - 예) 날씨 및 계절의 아이스크림 판매량에 대한 각각의 영향도와 상호작용을 확인
                    
                    (두 요인이 종속변수에 영향을 주는지 안 주는지 분석하는것)
                    
    
    ### 069 t-검정 예시

### 071 회귀 분석(Regression Analysis) 개요

1. 용어 정리
    1. 독립변수
        1. 다른 변수에 영향을 받지 않고 독립적으로 변화하는 수, 설명 변수라고도 한다
        2. 입력 값이나 원인을 나타내는 변수, $y=f(x)$에서 $x$에 해당하는 것을 의미한다
    2. 종속변수
        1. 독립변수의 영향을 받아 값이 변화하는 수로, 분석의 대상이 되는 변수이다
        2. 결과물이나 효과를 나타내는 변수, $y=f(x)$ 에서 $y$에 해당하는 것
    3. 전차(오차항)
        1. 계산에 의해 얻어진 이론 값과 실제 관측이나 측정에 의해 얻어진 값의 차이
        2. 모집단 용어- 오차(Error), 표본집단 용어- 잔차(Residual)
2. 회귀 분석(Regression Analysis) 
    1. 변수와 변수 사이의 관계를 알아보기 위한 통계적 분석 방법
    2. 독립변수의 값에 의해 종속변수의 값을 예측하기 위한 분석 방법
    3. 일반 선형회귀는 종속변수가 연속형 변수일 때 가능함
    4. 명목, 서열 척도- 이산형(범주형) /  구간,비율 척도 - 연속형 
3. 회귀 모형(Regression Model)
    1. 선형회귀 모형
        1. 종속변수 $y$와 한 개 이상의 독립변수 $x$와의 선형 상관관계를 모델링하는 회귀 분석 기법
        2. 한 개의 독립변수 : 단순선형회쉬 / 둘 이상의 독립변수 : 다중선형회귀
    2. 단순회귀모형 (독립변수 한개일 때)
        
        
        모집단
        
        $$
        Y_i=\beta_0+\beta_1X_i+\varepsilon_i \\ (i=1,2,...,n)
        $$
        
        $Y_i$ : 종속 변수
        
        $X_i$ : 독립 변수
        
        $\varepsilon_i$ : 오차항
        
        $\beta_0$ : 선형회귀식의 절편
        
        $\beta_1 :$ 기울기, 회귀계수(coefficient)

### 072 회귀 모형의 가정

1. 회귀 모형의 가정 `암기⭐️⭐️⭐️`
    1. 선형성 : 독립변수의 변화에 따라 종속변수도 변화하는 선형(linear) 모형이다
    2. 독립성 : 잔차와 독립변수의 값이 관련되어 있지 않다(Durbin-Watson 통계량 확인)
    3. 정규성(정상성, Normality) : 잔차항이 정규분포를 이뤄야 한다
    4. 등분산성 : 독립변수의 모든 값에 대해 잔차항들의 분포는 동일한 분산을 갖는다
    5. 비상관성 : 관측치들의 잔차들끼리 상관이 없어야 한다

### 073 데이터 정규성 검정

1. 데이터의 정규성 검정
    1. Normal Q-Q plot, Histogram, Shapiro Wilk test, Kolmogorov-Smirnov test,Anderson-Darling test 이 있다
    2. 그래프를 사용하는 Q-Q 플롯과 히스토그램은 통계적인 유의미성을 이야기할 수 없다
2. 데이터의 정규성 검정 방법
    1. Q-Q Plot
        1. 그래프를 그려서 정규성 가정이 만족되는지 시각적으로 확인하는 방법
        2. 대각선 참조선을 따라 값들이 분포하게 되면 정규성을 만족한다고 할 수 있음
    2. Histogram
        1. 구간별 돗수를 그래프로 표시하여 시각적으로 정규분포를 확인하는 방법
    3. Shapiro-Wilk Test(샤피로-윌크 검정)
        1. 데이터가 정규분포를 따르는 지 알아보는 검정
        2. 귀무가설 : 데이터가 정규분포를 따르지 않는다
        3. 대립가설 : 데이터가 정규분포를 따르지 않는다
        4. p-value가 0.05보다 크면 정규성을 가정하게 됨 
    4. Kolmogorov-Smirnov Test(K-S test 콜모고르프-스미르노프 검정)
        1. 표본 데이터가 특정 분포를 갖는 모집단에서 온 것인지 확인하는 적합도 검정
        2. 귀무가설 : 데이터는 특정 분포를 따른다
        3. 대립가설 : 데이터는 특정 분포를 따르지 않는다
        4. p-value가 0.05보다 크면 정규성을 가정하게 됨
    5. Anderson-Darling Test
        1. 데이터가 특정 분포를 따르는 지 확인하는 적합도 검정 방법
        2. 귀무가설 : 데이터는 특정 분포를 따른다
        3. 대립가설 : 데이터는 특정 분포를 따르지 않는다
        4. K-S Test를 수정한 적합도 검정으로 특정 분포의 꼬리에 K-S Test 보다 가중치를 더 두어 수행됨

1. 표본 회귀선의 유의성 검정 :
    1. 두 변수 사이에 선형관계가 성립하는지 검정하는 것
    2. 귀무가설 & 대립가설
        1. 귀무가설 : 회귀식의 기울기 계수 $\beta_1$ 은 0과 같다
        2. 대립가설 : 회귀식의 기울기 계수는 $\beta_1$ 은 0과 같지 않다
2. 회귀 모형의 해석
    1. 회귀 모형의 해석 방법
        1. 모형이 통계적으로 유의미한가? → F 통계량의 유의확률(p-value)로 확인 ⭐️
        2. 회귀계수들이 유의한가? → 회귀계수의 t값에 대한 유의확률로 확인 ⭐️
        3. 모형이 얼마나 설명력을 갖는가? → 결정 개수 ($R^2$) 확인 ⭐️
        4. 모형이 데이터를 잘 적합하고 있는가? → 잔차를 그래프로 그려 회귀 진단 수행
        5. 데이터가 모형 가정을 만족시키는가를 확인함 → 모형 가정 : 선형성, 독립성, 등분산성, 비상관성, 정상성
    2. F 통계량, p-value
        
        $$
        F\ \ 통계량 = 회귀제곱평균(MSR)\ /\ 잔차제곱평균(MSE)
        $$
        
        - 모델의 통계적 유의성을 검정하기 위한 검정통계량
        - F 통계량이 클수록 회귀 모형은 통계적으로 유의함
        - F 통계량에 대한 p-value < 0.05 일 때 통계적으로 유의함
    3. t 값, p-value
        
        $$
        t\ 값=\text{Estimate(회귀계수)}/ \text{Std.Error(표준오차)}
        $$
        
        - t 통계량이 크다는 것은 표준오차가 작다는 의미
        - t 통계량이 클수록 회귀계수가 유의함
        - t 값에 대한 p-value < 0.05일 때 통계적으로 유의함
    4. 결정계수 `구할수있어야함⭐️`
        
        $$
        결정계수(R^2)=회귀제곱합(SSR)\ /\ 총제곱합(SST), 1-(SST/SST)
        $$
        
        - 회귀식의 적합도를 재는 척도
        - 결정계수는 0-1 사이의 범위를 갖으며, 결정계수가 커질수록 회귀방정식의 설명력이 높아짐

### 076 다중공선성 및 변수 선택법

1. 다중공선성(Multicollinearity)의 이해
    - 모형의 일부 설명변수(=예측변수)가 다른 설명변수와 상관되어 있을 때 발생함
    - 중대한 다중공선성은 회귀계수의 분산을 증가시켜 불안정하고 해석하기 어렵게 만들기 때문에 문제가 됨
    - R의 vid 함수를 사용해 구할 수 있으며, 분산팽창지수(VIF, Variance Inflation Factor) 값이 10이 넘으면 다중공선성이 존재한다고 봄
    - 높은 상관관계가 있는 설명 변수를 모형에서 제거하는 것으로 해결할 수 있음
    - 설명변수를 제거하면 대부분 결정계수(R-squared)가 감소함
    - 단계적 회귀분석을 이용하여 제거할 수 있음
    - 주성분분석을 통해 선형관계에 있는 변수들을 사용해 선형관계가 없는 새로운 주성분을 생성함
    - 표본의 크기가 작을 때 변수 간의 상관관계를 더 정확하게 파악하기 어려울 수 있으며, 이로 인해 다중공선성을 예측하거나 다루는 것이 더 어려울 수 있음
2. 설명변수 선택 방법
    1. 설명 변수 선택 조합
        1. 모든 가능한 조합 :
            - 모든 가능한 독립변수들의 조합에 대한 회귀모형을 고려해 AIC, BIC의 기준으로 가장 적합한 회귀 모형 선택
            - AIC, BIC : 최소자승법의 R^2와 비슷한 역할을 하며, 적합성을 측정해주는 지표로, R^2는 큰값이 좋지만 , AIC, BIC는 작은 값이 좋음
        2. 후진제거법(Backward Elimination) :
            - 독립변수 후보 모두를 포함한 모형에서 출발해 제곱합의 기준으로 가장적은 영향을 주는 변수로부터 하나씩 제거하면서 더 이상 유의하지 않은 변수가 없을 때까지 설명변수를 제거하고, 이때 모형을 선택함
        3. 전진선택법 (Forward Selection
            - 절편만 있는 모델에서 출발해 기준 통계치를 가장 많이 개선시키는 변수를 차례로 추가하는 방법
        4. 단계별 선택법 (Stepwise Method)
            - 모든 변수가 포함된 모델에서 출발 또는 절편만 있는 모델에서 출발해 기준 통계치에 가장 도움이 되지 않는 변수를 삭제하거나, 모델에서 빠져 있는 변수 중에서 기준 통계치를 가장 개선시키는 변수를 추가하는 방법
    2. 회귀 모델에서 변수 선택을 위한 판단 기준
        
        Mallow’s Cp, AIC, BIC 등이 있으며 값이 작을 수록 좋음

### 077 과대적합(Overfitting)과 정칙화(Regularization)

1. 과대적합과 과소적합
    1. 과소적합 (Underfitting)
        1. 모델이 너무 단순하거나 훈련 데이터를 충분히 학습하지 못할 때 발생함
        2. 모델이 훈련 데이터에 대해 충분한 패턴을 학습하지 못하므로, 테스트 데이터에서도 성능이 좋지 않음
        3. 일반화 능력이 부족하며, 새로운 데이터에 대한 예측이 부정확할 수 있음
        4. 학습을 좀 더 진행하여 예측력을 높여 성능을 높여야 함
    2. 과대적합(Overfitting)
        1. 모델이 훈련 데이터에 지나치게 적합되어 훈련 데이터에 존재하는 잡음까지 학습한 경우에 발생함
        2. 훈련 데이터에 대한 예측은 우수하지만, 새로운 데이터나 테스트 데이터에서는 성능이 떨어질 수 있는 상태이다
        3. 모델이 훈련 데이터의 특정 패턴이나 노이즈에 과도하게 민감하게 반응하는 경향이 있음
        4. 학습을 방해하는 요소를 추가하여 과대적합을 해결해서 학습 데이터의 성능과 평가 데이터의 성능 차이가 많지 않도록 조정해야 함
2. 과대적합 문제와 해결 방법
    1. 주어진 샘플들의 설명변수와 종속변수의 관계를 필요이상 너무 자세하고 복잡하게 분석하여 발생함
    2. 샘플에 심취한 모델로 새로운 데이터가 주어졌을 때 제대로 예측해내기 어려울 수 있음
    3. 해결 방법으로 독립변수(Feature)의 개수를 줄이거나, 정칙화(Regularization)를 수행하는 방법이 있음
        
        ![image.png](/assets/ADsP/part3/image%2057.png)
        
        - 알고 있는 데이터에 대해서는 오차가 매우 적고, 새로운 입력 값을 사용했을 때 오차가 커질 수 있는 상태를 과대적합
3. 정칙화(=정규화, 규제, Regularizarion) 개념
    
    ![image.png](/assets/ADsP/part3/image%2058.png)
    
    - 베타값(회귀계수깂)에 제약을 주어 모델에 변화를 주는 방법
    - $\lambda$ 값은 정규화 모형을 조정하는 hyper parameter
    - $\lambda$ 값이 클수록 제약이 많아져 적은 변수가 사용되고, 해석이 쉬워지지만 Underfitting 됨
    - $\lambda$ 값이 작아질수록 제약이 적어 많은 변수가 사용되고, 해석이 어려워지며 Overfitting 됨
    - $\lambda$는 $\alpha$라는 기호로 사용되기도 함
4. $L1$, $L2$ norm
    1. norm : 선형대수학에서 벡터의 크기(magnitude) 또는 길이(length)를 측정하는 방법
        1. $L1$ norm (=Manhattan norm) : 베거의 모든 성분의 절대값을 더함
        2. $L2$ norm (=Euclidean norm) : 출발점에서 도착점까지의 거리를 직선거리로 측정함
            
            ![image.png](/assets/ADsP/part3/image%2059.png)
            
5. 정규화 선형회귀 모델(Regularized Linear Regression)
    1. 라쏘(Lasso) 회귀 (=능형 회귀)
        
        ![image.png](/assets/ADsP/part3/image%2060.png)
        
        > RSS : Regression Sum of Squares,  $\lambda$ : Lasso 규제의 강도,  $\beta_j$ : 회귀계수, $\beta_0$ : 절편
        > 
        - L1 norm을 패널티로 가진 선형회귀 방법
        - 회귀계수의 절대값이 클수록 더 큰 패널티를 부여함
        - 변수 선택이 사능하며, 변수 간 상관관계가 높으면 성능이 떨어짐
        - 오차 제곱의 합이 최소가 되게 하는 회귀계수와 절편을 찾는 동시에 회귀계수 절대값의 합이 최소가 되어야 함
        - 오차 제곱의 합(RSS) 대신 오차 제곱의 평균(MSE,Mean Squared Error)을 사용할 수 있음
        - 회귀계수의 모든 원소가 0이 되거나 0에 가깝게 됨
        - 회귀계수가 0이 되게 함으로써 그에 해당하는 특성들을 제외하는 것(=불필요 특성 제거)
        - 모델에서 가장 중요한 특성이 무엇인지 알게 되는 등 모델 해석력이 좋아짐
    2. 릿지(Ridge) 회귀
        
        ![image.png](/assets/ADsP/part3/image%2061.png)
        
        - L2 norm을 패널티로 가진 선형회귀 방법
        - 회귀계수의 제곱값이 클수록 더 큰 패널티를 부여함
        - 변수 선택이 불가능하며, 변수 간 상관 관계가 높아도 좋은 성능을 보임
        - Lasso는 가중치들이 0이 되지만, Ridge의 가중치들은 0에 가까워질 뿐 0이 되지는 않음
        - 특성이 많은데 특성의 중요도가 전체적으로 비슷하다면 Ridge가 좀 더 괜찮은 모델을 찾아줄 것임!
    3. 엘라스틱넷(ElasticNet) 회귀
        
        $$
        \frac{\Sigma_{i=1}^n(y_i-x_i^J\hat\beta)^2}{2n}+\lambda \left(\frac{1-\alpha}{2}\sum_{j=1}^m\hat\beta_j^2+\alpha\sum_{j=1}^m|\hat\beta_j| \right)
        $$
        
        - Regularization에 L1, L2 norm을 모두 사용함
        - L1,L2 norm을 모두 사용하기 때문에 변수 선택 가능, 변수 간 상관관계를 반영한 Regularization이 가능함
6. 편향과 분산 (Bias and Variance)
    
    ![image.png](/assets/ADsP/part3/image%2062.png)
    
    - 편향, 분산 : 학습 알고리즘이 갖는 두 가지 종류의 오류로 상충관계(Trade Off)임
    - 학습 모형이 유연할수록 모델 복잡도(Model Complexity)가 증가한다는 것을 의미하여, 편향이 낮고, 분산이 높은 상태가 됨
    - 편향 : 지나치게 단순한 모델로 인한 오류, Bias가 크면 과소적합(Underfitting)을 야기함
    - 분산 : 지나치게 복잡한 모델로 인한 오류, Variance가 크면 과대적합(Overfitting)을 야기함

### 078 회귀 모델 평가 지표

1. 회귀 모델 평가 지표 `⭐️식만 보고 뭔지 ⭐️`
    1. MAE (Mean Absolute Error) :
        1. 평균 절대값 오차
        2. MSE와 비교해 이상치에 대해 영향을 덜 받음
            
            $$
            MAE=\frac{\Sigma_{i=1}^n|y_i-\hat y_i|}{n}
            $$
            
    2. MSE (Mean Squared Error) :
        1. 평균 제곱 오차
        2. 1미만의 에러는 더 작아지고, 1이상의 에러는 커짐, 이상치에 민감
            
            $$
            MSE=\frac{\Sigma_{i=1}^n(y_i-\hat y_i)^2}{n}
            $$
            
    3. RMSE (Root Mean Squared Error) :
        1. 평균 제곱 오차 제곱근
        2. MSE의 값이 제곱근, MSE의 이상치에 대한 민감함 보완
            
            $$
            RMSE=\sqrt{MSE}
            $$
            
    4. MAPE (Mean Absolute Percentage Error) :
        1. 평균 절대 백분율 오차
        2. MSE, RMSE 평가지표의 경우 데이터 크기에 따라 오차의 크기가 달라져서 오차의 크기를 직관적으로 알 수 없다는 단점이 있으며, 이런 단점을 극복하기 위한 방법으로 MAPE가 사용됨
        3. 오차의 백분율로 구해지며, 단위 Scale이 되어 단위에 영향을 받지 않고, 음수 사용이 가능함
            
            $$
            MAPE=\frac{100\%}{n}\sum_{i=1}^n|\frac{y_i-\hat y_i}{y_i}|\quad
            \begin{align}
            y_i : \text{실제 값, actual value}\\ 
            \ \hat y_i:\text{예측 값, forecast value}
            \end{align}
            $$
            
2. MAPE 구하기 `⭐️계산 가능해야함⭐️`
    1. 다음 데이터에 대한 MAPE를 구하여 보자
        
        
        | actual | 1 | 2 | 4 | 8 |
        | --- | --- | --- | --- | --- |
        | forecast | 0.7 | 2.5 | 3.6 | 10 |
    2. MAPE : 각 항의 ‘실제값과 예측값의 차이/실제값’에 대한 절대값을 모두 합하여 데이터의 개수 n으로 나누고 100을 곱한 값
        
        $$
        MAPE=\frac{100\%}{n}\sum_{i=1}^n|\frac{y_i-\hat y_i}{y_i}|\quad
        \begin{align}
        y_i : \text{실제 값, actual value}\\ 
        \ \hat y_i:\text{예측 값, forecast value}
        \end{align}
        $$
        
        $$
        
        \begin{align}
        &n=4\\
        &|(1-0.7)/1|+|(2-2.5)|+|(4-3.6)/4|+|(8-10)/8|\\
        &0.3/1+0.5/2+0.4/4+2/8\\
        &0.3+0.25+0.1+0.25\\
        &MAPE=\frac{0.9}{4}=0.225*100\%=22.5\%
        \end{align}
        $$

### 080 데이터 스케일링 (Scaling) `식 암기`

- 데이터 스케일링이란?
    - 정규화와 표준화는 데이터 단위의 불일치 문제를 해결하는 방법
    - 분석에 사용되는 변수들에 사용 단위가 다를 때 데이터를 같은 기준으로 만드는 방법
1. 정규화(Normalization) :
    1. 값의 범위를 [0,1]로 변환하는 것
    2. 최대-최소 정규화(min-max normalization)
    
    $$
    X'=\frac{X-X_{min}}{X_{max}-X_{min}}\\
    X':\text{정규화 결과,}\ X:\text{정규화 대상,}\ X_{min}:\text{최솟값, }\ X_{max}:\text{최댓값} 
    $$
    
    - Min=0, max =100점일 경우 50점을 정규화 한 값은?
        
        = (50-0)/(100)=0.5
        
        ![image.png](/assets/ADsP/part3/image%2069.png)
        
2. 표준화(Standardization)
    1. 특성의 값이 표준정규분포를 갖도록 변환하는 것으로 평균을 0 표준편차를 1이 되로록 변환함
    
    $$
    Z'=\frac{X-\mu}{\sigma}\\
    Z':\text{표준화 결과},\ X:\text{표준화 대상},\ \mu:\text{X의 평균},\ \sigma:\text{X의 표준 편차}
    $$
    
    - 평균 80, 표준푠차 10일 경우 90점을 표준화한 값은?
        
        = (90-80)/10=1
        
        ![image.png](/assets/ADsP/part3/image%2070.png)
        
3. 구간화 (Binning) 연속형 → 범주형
    1. 데이터의 종류가 많은 경우, 구간으로 나누어 묶어 사용함
    2. 구간화의 개수가 감소하면 정확도가 낮아지고, 속도가 높아질 수 있음
    3. 성수기/비수기와 같이 임의의 구간을 만들 수 있음
        
        예) 날짜시간 → 월별, 일별, 요일별, 시간별 / 나이 → 연령별
        
4. 부호화(Encoding)
    1. 사용자가 입력한 문자나 기호들을 컴퓨터가 이용할 수 있는 것으로 바꾸는 것
    2. 범주형 데이터가 문자/문자열로 표현된 경우, 이산형 수치로 바꿔 사용함
        
        예) Label Encoding, One Hot Encoding, Binary Encoding 등
        
    
    ![image.png](/assets/ADsP/part3/image%2071.png)
    
5. 변환(Transformation)
    1. 큰값을 작게 만들기 위해 로그(log)변환, 제곱 루트 변환($\sqrt x$)을 사용하며, 오른쪽 꼬리 긴 분포를 정규분포로 만들 수 있음
    2. 작은 값을 크게 만들기 위해 지수(exp)변환, 제곱 변환($x^2$)을 사용하며, 왼쪽 꼬리 긴 분포를 정규분포로 만들수 있음

### 081 상관 분석

1. 상관계수의 이해
    1. 상관계수는 두 변수의 관련성의 정도를 의미, -1~1의 값으로 나타낸다
    2. 두 변수의 상관관계가 존재하지 않을 경우 상관계수=0
    3. 상관관계가 높다고 인과관계가 있다고 할 수는 없음
    4. 종류 : 피어슨, 스피어만
    5. 피어슨 상관계수:
        1. 두 변수 간의 선형적인 크기만 측정가능
    6. 스피어만 상관계수 :
        1. 두 변수 간의 비선형적인 관계도 나타낼수있음
    7. R의 `cor.test()` 함수를 사용해 상관계수 검정을 수행하고, 유의성 검정을 판단할 수 있음
    8. 이때 귀무가설 - ‘상관계수가 0이다‘ / 대립가설 - ’상관계수가 0이 아니다’
        
        ![image.png](/assets/ADsP/part3/image%2072.png)
        
2. 공분산(Covariance)
    1. 공분산 공식
        1. 모집단인 경우
            
            $$
            cov(x,y)=\frac{\sum_{i=1}^N(x-\mu_x)(y-\mu_y)}{N}\\
            \mu_x:x\text{의 평균},\mu_y:y의\ 평균, N: 데이터의\ 개수
            $$
            
        2. 표본인 경우
            
            $$
            cov(x,y)=\frac{\sum_{i=1}^n(x-\bar x)(y-\bar y)}{n-1}
            $$
            
    2. 2개 확률변수의 선형관계를 나타내는 값
    3. 하나의 변수가 상승하는 경향을 보일 때 다른 값도 상승하는 선형 상관성이 있다면 양의 공분산을 갖음
    4. 두 확률 변수 $x,y$가 독립이면 공분산 $cov(x,y)=0$ 이며, 관측 값들이 4면에 균일하게 분포되어 있다고 추정할 수 있음
        1. $cov(x,y)=0$ 이라고 해서 항상 두 확률변수가 독립인 것은 아님 
3. 피어슨 상관계수 (Pearson Correlation)
    1. x,y의 공분산을 x,y의 표준편파의 곱으로 나눈 값
    2. 공식
        1. 모집단의 경우 :
            
            $$
            corr(x,y)=\frac{cov(x,y)}{\sigma_x\sigma_y}
            $$
            
        2. 표본인 경우 :
            
            $$
            corr(x,y)=\frac{cov(x,y)}{s_xs_y}
            $$
            
    3. 대상자료는 등간척도, 비율척도 사용
    4. 두 변수 간의 선형적인 크기만 측정 가능
    
    <aside>
    <img src="https://www.notion.so/icons/link_gray.svg" alt="https://www.notion.so/icons/link_gray.svg" width="40px" />
    
    - 예1 ) 응답자1의 표준편차 2 응답자2의 표준편차 2, 두 응답자의 공분산 값 4
        - 피어슨 상관계수$(p)= corr(x,y)=\frac{cov(x,y)}{s_xs_y}=\frac{4}{2\times 2}=1$
    - 예2 ) 응답자의 키와 몸무게를 나타낸 표, 키와 몸무게의 피어슨 상관계수는?(단, 모집단에 대한 값을 구한다)
        
        
        | 응답자 ID | 키 | 몸무게 |
        | --- | --- | --- |
        | 1 | 165 | 65 |
        | 2 | 170 | 70 |
        | 3 | 175 | 75 |
        | 4 | 180 | 80 |
        | 5 | 185 | 85 |
        - $p=corr(x,y)=\frac{cov(x,y)}{\sigma_x \sigma_y}$
        - 키의 평균은 175, 표준편차는 $\sqrt {50}$ 이고, 몸무게의 평균은 75, 표준편차는 $\sqrt {50}$이다
        - 이때 키와 몸무게의 공분산은 50이다 ($cov(x,y)=\frac{\sum_{i=1}^n(x-\mu_x)(y-\mu_y)}{n}$)
        - 이를 피어슨 상관계수 식에 대입하면 $\frac{50}{\sqrt{50}\times \sqrt{50}}=1$
    </aside>
    
4. 스피어만 상관계수 (Spearman Correlation) `그래프 해석`
    1. 서열척도인 자료에서 사용가능
    2. 두 변수 간의 비선형적인 관계를 나타낼 수 있음
    3. 두 변수의 순위 사이의 통계적 의존성을 측정하는 비모수적인 척도, 연속형 외에 이산형도 가능
    4. 스피어만 상관계수는 원시 데이터가 아니라 각 변수에 대해 순위를 매긴 값을 기반으로 함
    5. 두 변수 안의 순위가 완전 일치하면 1, 완전 반대이면 -1
        
        예) 수학 잘하는 학생이 영어도 잘하는 것과 상관있는지 알아보는데 사용될 수 있음

### 082 차원(=독립변수의 개수) 축소

1. 차원 축소를 목표를 위해 개발된 분석 방법
    1. 주성분 분석 (Principal Component Analysis) :
        1. 상관관계가 있는 변수들을 선형 결합으로 상관관계가 없는 새로운 변수(주성분)를 만들고 분산을 극대화하는 변수로 축약함
        2. 공분산행렬 또는 상관계수 행렬을 사용해 모든 변수를 가장 잘 설명하는 주성분을 찾음
    2. 다차원 척도법 (Multi-Dimensional Scaling) :
        1. 개체들 사이의 상대적 거리 또는 유사성 정보를 활용하여 고차원 데이터를 저차원(2차원 혹은 3차원) 공간상에 점으로 표현하여 개체 사이의 군집을 시각적으로 표현하는 분석 방법
    3. 요인 분석 (Factor Analysis) : 
        1. 다차원 데이터 집합에서 숨겨진 구조를 파악하고 변수들 간의 상관관계를 이해하는 통계적 기법
        2. 주로 변수 간의 상관성을 고려하여 변수들을 몇 개의 잠재적인 요인(factor)으로 요약하려는 목적으로 사용됨
    4. 판별 분석 (Discriminant Analysis) :
        1. 여러 그룹 또는 범주 간의 차이를 감지하고 설명하는 통계적 기술
        2. 다차원 데이터에서 그룹 간의 차이를 최대화하고 그룹 내의 변이를 최소화하는 선형 결합을 찾아내려는 목적으로 사용됨
    5. 군집 분석 (Cluster Analysis) :
        1. 데이터를 유사한 특성을 갖는 그룹 또는 군집으로 나누는 비지도 학습 기법
        2. 데이터 간의 내재된 구조나 패턴을 찾아내기 위해 사용
    6. 정준상관 분석 (Canonical Correlation Analysis) :
        1. 둘 이상의 다른 변수 세트 간의 상관관계를 파악하고 이를 최대화하는 통계적 기술
        2. 다변량 데이터 분석에서 주로 사용되며, 변수 세트 간의 상호 관계를 밝히고 이를 통합하는 데 사용됨
2. 다차원 척도법 (MDS, Multi-Dimensional Scaling)
    - 개체들 사이의 상대적 거리 또는 유사성 정보를 활용하여 고차원 데이터를 저차원 공간상에 점으로 표현하여 개체 사이의 군집을 시각적으로 표현하는 분석 방법
    - 주요 목적은 객체 간 상대적 위치를 보존하면서 데이터를 시각화하거나 분석하기 쉬운 형태로 변환하는 것
    - 개체들의 거리는 유클리드 거리와 유사도를 이용하여 구함
    - 관측 대상의 상대적 거리의 정확도를 높이기 위해 적합 정도를 스트레스 값(Stress Value)로 나타내며, **$0$** 에 가까울수록 적합도가 좋음

### 083 주성분 분석 (PCA) `2,3 문제 나올수도`

1. 주성분 분석 (Principal Component Analysis) 
    1. 데이터를 분석할 때 변수의 개수가 많다고 모두 활용하는 것이 꼭 좋은 것은 아님
    2. 오히려 변수가 ‘다중공선성‘이 있는 경우 분석 결과에 영향을 줄 수도 있음
    3. 공분산행렬 또는 상관계수 행렬을 사용해 모든 변수들을 가장 잘 설명하는 주성분을 찾는 방법
    4. 상관계수가 있는 변수들을 선형 결합에 의해 상관관계가 없는 새로운 변수(주성분)을 만들고 분산을 극대화하는 변수로 축약함
    5. 주성분은 변수들의 선형결합으로 이루어져 있음
    6. 독립변수듫과 주성분과의 거리인 ’정보손실량’을 최소화하거나 분산을 최대화 함
    
    ![image.png](/assets/ADsP/part3/image%2077.png)
    
2. 주성분분석 할 때 고민해야 하는 것
    1. ⭐️공분산행렬과 상관계수행렬 중 어떤 것을 선택할 것인가?
    2. ⭐️주성분의 개수를 몇 개로 할 것인가?
    3. 주성분에 영향을 미치는 변수로 어떤 변수를 선택할 것인가?
3. 공분산 행렬(default) VS 상관계수 행렬
    1. 공분산 행렬은 변수의 측정단위를 그대로 반영한 것이고, 상관계수 행렬은 모든 변수의 측정단위를 표준화한 것이다
    2. 공분산행렬을 이용한 경우 측정 단위를 그대로 반영하였기 때문에 변수들의 측정 단위에 민감하다
    3. 주성분 분석은 거리를 사용하기 때문에 척도에 영향을 받는다(정규화 전후의 결과가 다르다)
    4. 설문조사처럼 모든 변수들이 같은 수준으로 점수화 된 경우 공분산행렬을 사용한다
    5. 변수들의 scale이 서로 많은 다른 경우에는 상관계수행렬(correlation matrix)을 사용한다
        - 주성분 분석에서 상관계수 행렬 사용
            - `prcomp(data,scale=TRUE)`, `princomp(data,cor=TRUE)`
            - Scale, cor 을 FALSE로 지정하거나 생략시 공분산 행렬이 사용됨
            - Prcomp,princomp의 결과는 같음
4. 주성분 개수 결정 기준
    1. 성분들이 설명하는 분산의 비율
        1. 누적 분산 비율을 확인하면 주성분들이 설명하는 전체 분산 양을 알 수 있음
        2. 누적 분산 비율이 70~90% 사이가 되는 주성분 개수 선택
    2. 고윳값(Eigenvalue)
        1. 분산의 크기(=중요도 기준)을 나타내며, 고윳값이 1보다 큰 주성분만 사용함
    3. 스크리 플롯 (Scree Plot)
        1. 고윳값을 가장 큰 값에서 가장 작은 값 순서로 정렬해 보여줌(1보다 큰 값 사용 또는 Elbow기법)

    1. 시계열 자료
        1. 시간의 흐름에 따라 관측된 데이터
        2. 시계열 분석을 위해서는 정상성을 만족해야 함
    2. 정상성(Stationary)
        1. 시계열의 수준과 분산에 체계적인 변화가 없고, 주기적 변동이 없다는 것
        2. 미래는 확률적으로 과거와 동일하다는 것
2. 정상 시계열의 조건
    1. 평균은 모든 시점(시간 $t$)에 대해 일정하다
        
        $$
        E(x_t)=\mu
        $$
        
    2. 분산은 모든 시점(시간 $t$)에 대해 일정하다
        
        $$
        Var(x_t)=\sigma^2
        $$
        
    3. 공분산은 시점(시간 $t$)에 의존하지 않고, 단지 시차에만 의존한다
        
        $$
        Cov(x_{t+h},x_t)=\gamma_h
        $$

1. 정상시계열 전환
    1. 정상 시계열 전환
        1. 비정상시계열 자료는 정상성을 만족하도록 데이터를 정상시계열로 만든 후 시계열 분석을 수행한다
        2. 평균이 일정하지 않은 경우 : 원계열에 차분 사용
        3. 계절성을 갖는 비정상시계열 : 계절 차분 사용
        4. 분산이 일정하지 않은 경우 : 원계열에 로그 변환을 사용
    2. 차분(Differencing)
        
        현 시점의 자료 값에서 전 시점의 자료 값을 빼 주는 것을 의미
        
        $$
        Y_t'=Y_t-Y_{t-1}
        $$
        
    3. 로그 변환 
        
        시계열 데이터 값에 로그 함수를 취함으로써 데이터의 변동을 조절함(큰 값의 영향을 줄이고 작은 값의 영향을 높일 수 있음)
        
        $$
        Y_t'=log(Y_t)
        $$

### 085 시계열 모형, 분해 시계열

1. 시계열 모형의 종류
    1. 자기회귀 모형 (AR 모형, Auto Regressive Model)
        1. AR(p) : 현시점의 자료를 p시점 전의 유한 개의 자기 자신의 과거값을 사용하여 설명
        2. 백색 잡음의 현재 값($\epsilon_t$)과 자기 자신의 과거 값($y_{t-p}$)의 선형 가중 값으로 이루어진 정상 확률 모형
        3. 현 시점의 시계열 자료에 과거 1시점 이전의 자료만 영향을 주면 이를 1차 AR 모형이라고 하고 AR(1)로 표기함
        4. 수식
            
            $$
            y_t=c+\phi_1y_{t-1}+ \phi_2y_{t-2}+\cdots+ \phi_py_{t-p}+\epsilon_t
            \\c:\text{drift,모델의 평균 또는 추세를 나타내는 상수}\\
            \epsilon_t :\text{백색 잡음의 현재값}\\
            y_{t-1},y_{t-2},\cdots: \text{자기 자신의 과거 값}
            $$
            
    2. 이동평균 모형(MA 모형, Moving Average Model)
        1. MA(q) : 과거 q 시점 이전 오차들에서 현재항의 상태를 추론한다
        2. 최근 데이터의 평균을 예측치로 사용하는 방법, 각 과거치는 동일 가중치가 주어짐
        3. 현시점의 자료가 유한 개의 과거 백색잡음의 선형결합으로 표현되었기 때문에 항상 정상성을 만족함
        4. 수식
            
            $$
            y_t=c+\epsilon_t+\theta_1\epsilon_{t-1}+ \theta_2\epsilon_{t-2}+\cdots+ \theta_q\epsilon_{t-q},\ \epsilon_t \sim N(0,\sigma^2)\\
            c:\text{모델의 평균을 나타내는 상수}\\
            \epsilon:\text{백색 잡음(t는 현재,t-1은 과거 시점을 의미함}
            $$

        백색잡음(White Noise)
        
        - 시계열 자료 중 자기상관이 전혀 없는 특별한 경우
        - 시계열의 평균이 0, 분산이 일정한 값, 자기공분산이 0인 경우
        - 현재 값이 미래 예측에 전혀 도움이 되지 못함
        - 회귀 분석의 오차항과 비슷한 개념
        </aside>

        이동 평균법(Moving Average)
        
        - 일정 기간의 관측치에 모두 동일 가중치를 부여하여 이동 평균을 계산하는 방법
        - 추세, 주기를 측정하기 위해 사용함, 계절 성분과 불규칙 성분을 제거함
        
        $$
        F_t=\frac{Y_{t-1}+Y_{t-2}+\cdots+Y_{t-n}}{n}
        $$
        
        $F_t$ : $t$ 시점의 예측치, $Y_t$ : $t$ 시점의 관측치, $n$ : 이동평균 기간
        
        </aside>

        지수평활법(Exponential Smoothing)
        
        - 이동평균법의 종류로, 최근 관측치에 더 높은 가중치를 부여하는 방법
        - 최신의 변동사항을 민감하게 반영함
        - 최근 시점에 큰 가중치, 과거 시점으로 갈수록 가중치를 지수적으로 줄여 나감
        - 자수평활을 사용하여 얻은 예측치는 가장 최근 관측값과 이전 예측값의 가중 평균
        </aside>
        
    3. 자기회귀 누적 이동평균 모형(ARIMA 모형, Auto-Regressive Integrated Moving Average)
        1. 현재와 추세 간의 관계를 정의한 것으로, 많은 시계열 자료가 ARIMA 모형을 따른다
        2. ARIMA 모형은 비정상 시계열 모형이며 차분/변환을 통해 AR,MA,ARMA 모형으로 정상화할 수 있다
        3. ARIMA(p,d,q)에서 p는 AR 모형 차수, d는 차분, q는 MA 모형 차수를 의미한다
            
            예) ARIMA(1,2,3)이라면 2번 차분해서 ARMA 모형이 될 수 있다
            
        4. 차분을 나타내는 d 차수만큼 차분하면 정상 시계열이 될 수 있다
            
            예) ARIMA(0,1,3) : IMA(1,3) 모형이고, 이것을 1번 차분하면 MA(3) 모형이 됨
            
            예) ARIMA(2,3,0) : ARI(2,3) 모형이고, 이것을 3번 차분하면 AR(2) 모형이 됨
            
2. 자기 상관 함수(ACF), 부분 자기 상관 함수(PACF)
    1. 자기 상관 함수 (ACF ; Auto-Correlation Function)
        
        $$
        ACF(k)=\frac{\sum_{t-1}^{N-k}(y_t-\bar y)(y_{t+k}-\bar y)}{\sum_{t=1}^N(y_t-\bar y)^2}
        $$
        
        - 시계열 데이터의 자기 상관성을 파악하기 위한 함수
        - 시계열 관측치 y_t 와 y_{t-k}간의 상관계수를 k의 함수 형태로 표시한 것 (k:시간단위)
        - k가 커질수록 ACF는 0으로 수렴한다
        
        $$
        -1\le \text{autocorr}(Y_t,Y_{t-k})\le 1
        $$
        
        - 정상 시계열과 비정상 시계열
            - 정상 시계열 : 상대적으로 빠르게 0에 수렴함
            - 비정상 시계열 : 천천히 감소하여, 종종 큰 양의 값을 가짐
    2. 부분 자기 상관 함수 (PACF ; Partial ACF)
        
        $$
        PACF(k)=Corr(e_t,e_{t-k})
        $$
        
        - Y_t와 Y_{t-k} 중간에 있는 값들의 영향을 제외시킨 Y_t와 Y_{t-k} 사이의 직접적 상관관계를 파악하기 위한 함수
        - 시차가 다른 두 시계열 데이터 간의 상관관계를 의미한다

3. 분해 시계열 ⭐️
    1. 분해 시계열 : 시계열에 영향을 주는 일반적인 요인을 시계열에서 분리해 분석하는 방법
    2. 분해시계열 분해 요인 (`종류는 반드시 암기!`)
        1. 추세 요인 (Trend Factor)
            - 자료의 그림을 그렸을 때 그 형태가 오르거나 내리는 등 자료가 어떤 특정한 형태를 취할 때
        2. 계절 요인 (Seasonal Factor)
            - 계절에 따라, 고정된 주기에 따라 자료가 변화하는 경우
                
                예) 1년 중 특정할 때, 1주일의 특정 요일에 나타나는 요인
                
        3. 순환 요인 (Cyclical Factor)
            - 물가상승률, 급격한 인구 증가 등의 이유로 알려지지 않는 주기를 가지고 자료가 변화하는 경우
        4. 불규칙 요인
            - 위 세 가지 요인으로 설명할 수 없는 회귀분석에서 오차에 해당하는 요인에 의해 발생하는 경우

4. 시계열 분해 그래프
    1. 시계열 분해 그래프 : 시계열 분해 그래프로 “관측치, 추세, 계절성, 잔차”를 확인할 수 있음
        
        ![image.png](/assets/ADsP/part3/image%2087.png)
        
        - 추세 :
            - 데이터의 장기적인 경향이나 움직임을 나타냄
            - 추세는 시간이 지남에 따라 증가하거나 감소할 수 있음
        - 계절성 :
            - 데이터에 주기적으로 반복되는 패턴이 있을 경우 해당 부분을 나타냄
            - 계절성은 일정한 주기(예:연간, 월간)를 가지며 주기마다 패턴이 반복됨
        - 잔차 :
            - 추세와 계절성으로 설명되지 않는 나머지 부분으로, 랜덤한 불규칙성을 나타냄
            - 위의 그래프에서는 -5~5 사이의 잔차가 있는 것을 확인할 수 있음
5. 시계열 데이터의 분석 절차  `순서 확인`
    1. 시계열 그래프 그리기 :
        1. 분석의 시작은 시계열 데이터를 시각적으로 탐색하는 것
        2. 시계열 그래프를 그려서 데이터의 패턴,추세,계절성, 이상치 등을 시각적으로 확인함
    2. 추세와 계절요인 파악 및 제거하기 :
        1. 데이터에서 추세와 계절성을 식별하고 이것을 제거하여 시계열 데이터를 안정적인 상태로 만듦
    3. 잔차 예측하기 :
        1. 예측된 잔차를 모델링하여 다음 시점의 잔차를 예측하는 모델을 개발함
        2. 주로 ARIMA, Exponential Smoothing 등의 시계열 모델을 사용할 수 있음
    4. 예측된 잔차와 계절성을 재반영하여 예측하기 :
        1. 모델을 사용하여 예측된 잔차를 다시 추세와 계절성을 반영하여 최종 예측값을 생성함
        2. 이 예측값은 원래 데이터의 패턴을 반영하면서 미래 시점의 값을 예측하는 데 사용함
